DeepSeek, ClaudeCode e Codex

PROMPT USADO:

Contexto

você é um desenvolvedor front-end iniciante em HTML, CSS e JavaScript puro. Sua tarefa é criar uma aplicação web simples para um trabalho acadêmico de progrmação

Objetivo

Desenvolver uma aplicação web chamada "Minha Pokédex de Capturas".

A aplicação simula o registro dos Pokémon capturados por um jogador durante sua jornada em um jogo Pokémon.

Quando um jogador inicia sua aventura, ele pode escolher apenas um Pokémon inicial. Por isso, a lista deve começar com os três Pokémon iniciais da região de Kanto:

Charmander
Bulbasaur
Squirtle

Ao utilizar o sistema, o usuário deverá remover os dois Pokémon que não escolheu e manter apenas seu inicial. Conforme avança em seu save, poderá registrar novos Pokémon capturados.

Tecnologias Permitidas
HTML5
CSS3
JavaScript puro (Vanilla JavaScript)

Não utilizar:

Frameworks
Bibliotecas externas
APIs externas
Banco de dados
Local Storage
Requisitos Funcionais
Tela de Login

A aplicação deve iniciar exibindo apenas a tela de login.

Credenciais válidas:

Usuário: aluno
Senha: fiap2025

Validações:

Não permitir campos vazios.
Exibir mensagens de erro visíveis na tela.
Caso o login esteja correto, ocultar a tela de login e exibir a aplicação.
CRUD de Pokémon

Os dados devem ser armazenados em um array de strings.

A lista inicial deve conter exatamente:

Charmander
Bulbasaur
Squirtle

O usuário deve conseguir:

Adicionar Pokémon ao final da lista

Representa um Pokémon recém-capturado.

Adicionar Pokémon ao início da lista

Representa um Pokémon colocado como principal no time.

Visualizar Pokémon

Todos os Pokémon devem ser exibidos dinamicamente na tela.

Editar Pokémon

Permitir corrigir ou alterar o nome de um Pokémon específico.

Remover Pokémon

Permitir remover um Pokémon específico da lista.

A remoção deve ocorrer pela posição (índice) do item e não pelo valor.

Validações Obrigatórias
Não permitir adicionar Pokémon com nome vazio.
Exibir mensagens de erro visíveis ao usuário.
Ao editar, se o usuário cancelar ou informar um valor vazio, o Pokémon deve permanecer inalterado.
Campos de login não podem ser enviados vazios.
Toda alteração deve atualizar automaticamente a interface.

Criar uma interface inspirada no universo Pokémon utilizando apenas CSS.

Sugestões:

Título: "Minha Pokédex de Capturas"
Cores inspiradas em Pokébola (vermelho, branco e preto)
Layout simples e organizado
Botões com nomes relacionados ao tema Pokémon

Exemplos:

"Capturar Pokémon" (adicionar ao final)
"Adicionar ao Time Principal" (adicionar ao início)
"Editar"
"Liberar Pokémon" (remover)
